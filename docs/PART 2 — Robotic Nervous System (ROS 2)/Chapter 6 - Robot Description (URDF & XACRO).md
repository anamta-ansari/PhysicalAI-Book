# Chapter 6: Robot Description (URDF & XACRO)

## URDF basics
URDF (Unified Robot Description Format) defines robot models in XML.

## Creating a humanoid URDF
Design links, joints, sensors, and actuators to represent a humanoid robot.

## Joints, links, sensors, actuators
- **Links**: Rigid bodies of the robot
- **Joints**: Connections allowing movement
- **Sensors**: Perception devices
- **Actuators**: Motors enabling motion

## Xacro for modular robot building
Xacro (XML Macros) simplifies building modular and reusable URDF files.
